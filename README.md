Portfolio/Blog built using ReactJS, TailwindCSS, and powered by Sanity.io

## Run Locally

Clone the project

```bash
  git clone https://github.com/carloswhite/portfolio-app.git
```

Go to the project directory

```bash
  cd portfolio-app
```

Install dependencies

```bash
npm install 
```

Start the server

```bash
  npm run start
```
